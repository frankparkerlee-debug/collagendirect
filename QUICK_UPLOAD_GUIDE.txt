================================================================================
HOW TO UPLOAD FILES
================================================================================

STEP 1: ID CARD
───────────────
1. Click [Choose File] under "ID Card / Driver's License"
2. Select a PDF or image file from your computer
3. File uploads automatically
4. You'll see: "ID Card uploaded successfully!"
5. Page reloads - you'll now see a green ✓ checkmark

STEP 2: INSURANCE CARD
──────────────────────
1. Click [Choose File] under "Insurance Card"
2. Select a PDF or image file
3. File uploads automatically
4. You'll see: "Insurance Card uploaded successfully!"
5. Page reloads - you'll now see a green ✓ checkmark

STEP 3: AOB (ASSIGNMENT OF BENEFITS)
─────────────────────────────────────
1. Click [Generate & Sign AOB]
2. Click "OK" to confirm
3. AOB is auto-generated
4. You'll see: "AOB generated and signed successfully!"
5. Page reloads - you'll now see "✓ Signed"

DONE! All three should show green ✓ checkmarks

================================================================================
TROUBLESHOOTING
================================================================================

Can't see the upload section?
→ Make sure you clicked "View / Edit" on a patient
→ Scroll down - it's below the patient details

Upload doesn't work?
→ Check file is under 25MB
→ Use PDF or JPG/PNG format
→ Check browser console (F12) for errors

Files don't show after upload?
→ Manually refresh the page
→ Check /tmp/php-server.log for errors

Still getting "files must be on file" error?
→ Make sure all THREE files show green ✓
→ ID Card ✓
→ Insurance Card ✓
→ AOB ✓
